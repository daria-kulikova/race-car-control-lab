# Chronos and CRS Datasets
Together with the Chronos hardware and firmware and the CRS software, we also share the datasets collected to perform system identification and the evaluation of the state estimator used in the following paper:

> Sabrina Bodmer, Lukas Vogel, Simon Muntwiler, Alexander Hansson, Tobias Bodewig, Jonas Wahlen, Melanie N. Zeilinger, and Andrea Carron, "Optimization-Based System Identification and Moving Horizon Estimation Using Low-Cost Sensors for a Miniature Car-Like Robot", ArXiv 2024

We believe that these datasets can become a useful benchmark for nonlinear system identification and estimation.

## Dataset for System Identification 
The dataset used for system identification, can be found [here](https://gitlab.ethz.ch/ics/crs/-/tree/main/software/src/scripts/estimator_comparison?ref_type=heads). 

## Dataset for State Estimation Evaluation 
The dataset used for evaluating the state estimation can be found [here](https://gitlab.ethz.ch/ics/crs/-/tree/main/software/src/ros4crs/tools/opt_sys_id?ref_type=heads).
