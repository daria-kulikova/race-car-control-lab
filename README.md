# CRS Overview
[![pipeline status](https://gitlab.ethz.ch/ics-group/projects/andrea/crs-2.0/badges/dev/pipeline.svg)](https://gitlab.ethz.ch/carrona/crs/commits/dev)

In this repository, you can find all the electronics, firmware, and software to run Chronos and CRS. Chronos is a car-like 1/28th scale robot with customized open-source
electronics. CRS isan open-source software framework for control and robotics.You can find more information in the related folders.
